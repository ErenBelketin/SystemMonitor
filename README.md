# System Monitor Overlay

A sleek and modern system resources monitoring overlay for Windows, built with **Electron**. It provides real-time data on CPU, GPU, RAM, and Network usage, similar to MSI Afterburner but with a focus on aesthetics.

## 🚀 Features

- **Real-time Monitoring:** Tracking CPU load, temperature, GPU usage, VRAM, and Network speeds.
- **Glassmorphism Design:** Modern UI with blur effects and sleek animations.
- **Always on Top:** Stays visible while you are working or gaming.
- **Dynamic Chart:** Visualizing performance trends.

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ErenBelketin/SystemMonitor.git
   cd SystemMonitor
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the application:**
   ```bash
   npm start
   ```
   *or use the provided `baslat.bat` file.*

## 📦 Building

To generate a portable executable for Windows:
```bash
npm run build
```
The output will be located in the `dist` folder.

## 📝 Technologies Used

- **Electron:** Cross-platform desktop apps.
- **Systeminformation:** Node.js system data library.
- **HTML/CSS/JS:** Vanilla web technologies for the UI.
